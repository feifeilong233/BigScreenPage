## 基于Web3D的沥青路面性能试验平台

#### 介绍
springboot2.6.x+vue3-antdesign-vite架构，实现了系统管理模块、权限控制模块（菜单权限、功能按钮权限、数据权限）、代码生成器（单表、树表）、quartz动态定时器等功能。
